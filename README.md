# VMDNKO - Преміальний Сайт Настінних Панелей та Дверей

Сучасний преміальний багатосторінковий сайт для бренду VMDNKO, який займається продажем настінних панелей та дверей.

## 🎨 Особливості Дизайну

- **Преміальний темний дизайн** з акцентами золотого та бежево-піщаного кольорів
- **Grid layout** та card-based дизайн
- **Hover ефекти** та плавні анімації
- **М'які тіні** для глибини
- **SVG іконки** (без emoji)
- **Сучасні шрифти**: Montserrat та Inter
- **Багато "повітря"** (whitespace) для елегантного вигляду

## 📄 Структура Сайту

### 1. Головна (index.html)
- Hero-блок з великим заголовком та CTA кнопкою
- Блок переваг (3 cards)
- Категорії продукції (grid)
- Популярні товари (cards з hover ефектами)
- Коротко про компанію
- CTA-блок

### 2. Каталог (catalog.html)
- Список категорій (Настінні панелі, Двері)
- Фільтри (тип, матеріал, колір)
- Grid товарів з hover ефектами

### 3. Сторінка товару (product.html)
- Галерея зображень
- Назва товару та опис
- Характеристики
- Кнопка "Залишити заявку"
- Блок "Схожі товари"

### 4. Про нас (about.html)
- Історія бренду VMDNKO
- Цінності компанії
- Фото виробництва
- Статистика

### 5. Контакти (contacts.html)
- Контактна інформація
- Google Map (вбудована карта)
- Форма зворотного зв'язку з валідацією
- Модальне вікно після відправки

## 🚀 Технології

- **HTML5** - Семантична верстка
- **CSS3** - Сучасні CSS змінні, Flexbox, Grid
- **JavaScript** - Інтерактивність та валідація форм
- **Google Fonts** - Montserrat та Inter

Сайт повністю адаптивний для:

## ⚙️ Як Використовувати

1. Відкрийте `index.html` у браузері для перегляду головної сторінки
2. Навігація між сторінками через меню в хедері
3. Для локального розгортання використовуйте будь-який статичний сервер:
   - VS Code: Live Server extension
   - Python: `python -m http.server`
   - Node.js: `npx serve`

## 📝 Валідація Форми

Форма на сторінці контактів включає валідацію:
- Ім'я (обов'язкове)
- Телефон (обов'язковий, формат +380...)
- Email (обов'язковий, валідація формату)
- Повідомлення (обов'язкове)

Після успішної відправки з'являється модальне вікно з повідомленням про успіх.

## 🖼️ Зображення

На даний момент використовуються CSS градієнти як плейсхолдери. Для продакшену замініть їх на реальні зображення:
- Додайте зображення в папку `images/`
- Оновіть CSS класи `.product-bg-*`, `.category-bg-*`, `.production-bg-*` з реальними URL зображень

## 🔧 Налаштування

### Зміна контактів
